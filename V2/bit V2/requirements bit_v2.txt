# bit_v2 wallet — hash-pinned dependencies (D1: cryptographic hash verification).
#
# All direct packages are pinned with == and --hash=sha256:... so that
# `pip install --require-hashes` verifies the cryptographic integrity of each
# distribution before installation.  This eliminates a class of supply-chain
# attacks where a compromised PyPI mirror or CDN serves a modified package.
#
# Hashes computed on macOS (Apple Silicon / x86_64); wheels are platform-
# specific for coincurve and cryptography.  Re-generate for your platform:
#
#     pip download bit==0.8.0 coincurve==21.0.0 cryptography==46.0.0 requests \
#         --no-deps -d /tmp/deps
#     pip hash /tmp/deps/*.whl /tmp/deps/*.tar.gz
#
# To install with hash verification:
#     pip install --require-hashes -r requirements_v2.txt
#
# Dependency tree:
#   bit ── coincurve ── (libsecp256k1 via cffi)
#       └─ requests
#   cryptography  (AES-GCM / PBKDF2-HMAC-SHA256 for keystore)
#
# pip-audit is an optional scanning dependency (D2); install separately:
#   pip install pip-audit
